# 🏨 Belka Golf Residence - Belek, Antalya

Modern ve responsive web sitesi - Belka Golf Residence lüks apart tatil kompleksi için tasarlanmış profesyonel tanıtım sitesi.

## 📋 Proje Hakkında

Belka Golf Residence, Belek'in kalbinde, golf sahası yanında konumlanmış ve havuzlu, tam donanımlı lüks apart daireler sunmaktadır. Bu proje, işletmenin çevrimiçi varlığını güçlendirmek için geliştirilmiştir.

### ✨ Özellikler

- **📱 Responsive Tasarım** - Mobil, tablet ve masaüstü cihazlara tam uyumlu
- **🌍 Çok Dil Desteği** - TR, EN, DE, RU dil seçenekleri
- **🎯 SEO Optimized** - Schema.org JSON-LD yapılandırılmış veri
- **✨ Smooth Animasyonlar** - CSS animasyonları ve transitions
- **📅 Rezervasyon Sistemi** - Tarih seçimi ve oda türü seçme
- **🖼️ Galeri** - CSS Grid masonry galeri düzeni
- **⭐ Platform Puanları** - Booking, Airbnb, Google vb. derecelendirmeler
- **📍 Harita Entegrasyonu** - Konum bilgileri
- **💬 Müşteri Yorumları** - 5 yıldızlı testimoniallar
- **📞 WhatsApp Entegrasyonu** - Hızlı iletişim FAB

## 🚀 Kurulum & Çalıştırma

### 1. Projeyi Clone Et
```bash
git clone https://github.com/kullanici-adi/belka-golf-website.git
cd belka-golf-website
```

### 2. Live Server ile Çalıştır (VS Code)
- VS Code'u aç ve `index.html` dosyasını aç
- "Go Live" butonuna tıkla (sağ alt köşede)
- Tarayıcıda http://localhost:5500 adresinde açılacak

### 3. HTTP Server ile Çalıştır (Python)
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```
Ardından http://localhost:8000 adresine git

### 4. Node.js http-server Kullan
```bash
npm install -g http-server
http-server
```
